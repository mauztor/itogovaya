# Система учёта расходов и доходов текущего и капитального ремонта жилой недвижимости

Это веб-приложение на Flask для учёта доходов и расходов, связанных с ремонтом жилой недвижимости.

## Установка
1. Установите зависимости:
    pip install -r requirements.txt
2. Запустите приложение:
    python run.py
3. Откройте браузер и перейдите по адресу http://127.0.0.1:5000/.

## Функционал
- Управление категориями финансовых операций.
- Добавление финансовых операций.
- Просмотр списка операций.
- Анализ данных (баланс, расходы по категориям).
- Визуализация данных.
